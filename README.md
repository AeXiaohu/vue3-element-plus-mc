# vue3-element-plus-mc

## 介绍

基于vue3+element-plus的 Minecraft Server 网站

可供我的世界服务器网站参考学习

[在线预览地址](https://aexiaohu.github.io/vue3-element-plus-mc/)


## 项目安装
```
npm install
```

### 编译开发
```
npm run serve
```

### 编译生产
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Vue CLI  自定义配置
See [Configuration Reference](https://cli.vuejs.org/config/).
